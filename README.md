# CLI Calculator (Python)

A simple and robust command-line calculator built to practice core Python fundamentals and clean input handling.

## Features
- Supports basic operations: `+`, `-`, `*`, `/`, `**` (power)
- Validates numeric input (supports decimals, also accepts comma as decimal separator)
- Handles division by zero safely
- Runs in a loop until the user exits

## Tech Stack
- Python 3 (no external dependencies)

## Run Locally
```bash
python src/main.py
